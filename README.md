<h1 align="center">Tic Tac Toe</h1>

[![Site preview](/public/image.png)](https://pinuya.site)
⭕ a simple and beautiful tic-tac-toe game to play with your friends ❌

## Install & run

```bash
bun install
```

Once it's done start up a local server with:

```bash
bun dev
```
